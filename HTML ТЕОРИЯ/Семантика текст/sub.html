- Подстрочный текст для формул. - Чаще всего sub используется для записи химических или математических формул. - Можно использовать тег sub внутри тега var для вывода индекса переменных, например, элементов массива или матрицы. - Можно вкладывать друг в друга, тогда на каждом шаге текст будет спускаться ниже, а его размер будет уменьшаться.
