- Блок цитат. - Цитата может быть длинной, состоять из нескольких абзацев. - Для цитаты, которая скрывается в дебрях текста (внутристрочная) есть свой отдельный тег — q. - Сам этот тег только показывает, где цитата находится. - Внутри неё нужно разметить текст подходящими по смыслу тегами, например p. - Указать источник, откуда цитата была взята. Для этого есть cite. - В теге cite или в атрибуте cite указывается именно источник цитаты (название источника цитаты, либо ссылка на него), а не её автор.
