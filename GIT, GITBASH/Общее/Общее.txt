# в git есть 3 состояния файла:
- отслеживаемые
- неотслеживаемые (untracked, это новые файлы)
- игнорируемые

# отслеживаемые:
- изменённые (modified, это файлы, которые вы добавили командой git add, но после добавления изменили)

# git add добавляет файлы и/или их состояния во «временное хранилище», его называют index, или staged
# git commit фиксирует текущее состояние, очищая staged

# Для «Гит» пустая папка — это отсутствующая папка

# HEAD — это указатель на последний коммит.
# Файл HEAD содержит ссылку на служебный файл: refs/heads/main.

# какая версия установлена
git --version
