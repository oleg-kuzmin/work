git add добавляет файлы во «временное хранилище», его называют index, или staged

untracked — не отслеживаемые, это новые файлы, вы их ещё не добавляли командой git add
modified — изменённые, это файлы, которые вы добавили командой git add, но после добавления изменили

Для «Гит» пустая папка — это отсутствующая папка

HEAD — это указатель на последний коммит.
Файл HEAD содержит ссылку на служебный файл: refs/heads/main.
