- Пример, образец, результат выполнения программы. - Используется для оформления вывода данных, которые показываются пользователю в результате выполнения программы. - Содержимое этого элемента воспринимается устройствами как простой текст. - Подстрочный текст для формул. - Чаще всего sub используется для записи химических или математических формул. - Можно использовать тег sub внутри тега var для вывода индекса переменных, например, элементов массива или матрицы. - Можно вкладывать друг в друга, тогда на каждом шаге текст будет спускаться ниже, а его размер будет уменьшаться. - Надстрочный текст для формул. - Чаще всего sup используется для вывода степеней в математических формулах. - Можно использовать тег sup внутри тега var. - Можно вкладывать друг в друга, тогда на каждом шаге текст будет подниматься вверх, а его размер будет уменьшаться. - Смысловое разделение блоков. - Декоративные линии лучше свёрстать как псевдоэлементы или т.п. - Помогает отделить независимые друг от друга блоки — например, подразделы в статье. - Пользоваться только, когда без разделителя информация теряет смысл или в ней становится трудно разобраться. - Перед заголовком не стоит ставить hr — заголовок сам по себе является тематическим и визуальным разделителем.
