В ходе работы программы вам надо указать необходимое количество итераций:

Укажите значение переменной TmplRepeat: 

int main()
  {
    int age;
    double weight;
    std::cout << "Укажите возраст: ";
    std::cin >> age;
    std::cout << "Укажите вес: ";
    std::cin >> weight;
  }

Для вставки текста нажмите Ctrl + V.
