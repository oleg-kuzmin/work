- Address (Адрес, контактные данные)
- Article (Статья, товар, карточка пользователя)
- Aside (Блок с дополнительным содержимым)
- Blockquote (Большая цитата) - Body (Тело страницы)
- Division (Раздел, пустой блок)
- Footer (Нижний колонтитул, подвал страницы)

- Heading (Заголовки на странице)
- Header (Верхний колонтитул, шапка страницы)
- Horizontal rule (Горизонтальная линейка) - HyperText Markup Language (Родительский блок страницы)
- Main (Основной контент страницы)
- Navigation (Навигация на странице)

- Paragraph (Параграф, абзац) 
               - Preformatted text (Предварительно отформатированный текст)

       - Section (Секция, логический блок страницы)



             - Abbreviation (аббревиатура или акроним)
               - Definition (определение, описание, вывод терминов или расшифровка сокращений)

             - Time (время, разметка для даты и времени)

                   - Bold (жирный, выделение важного контента, но не самого)

                      - Break (перенос строки)
               - Delete (удалить, отображает значимый удалённый контент)
                 - Emphasis (акцент, добавляет смысловой акцент)
                   - Italic (курсив, изменение голоса или настроения, ремарка)
               - Insert (вставить, отображает значимый добавленный контент)
             - Marker (маркер, выделение маркером)
                   - Strikethrough (зачеркнутый, неактуальный текст, старая цена товара)
           - Small (маленький, пояснения, комментарии)
             - Span (интервал, выделение части текста в блоке)
         - Strong (сильный, выделение важного контента)
                   - Underline (подчеркнутый, текст с ошибками)

             - Code (оформление программного кода)
             - Data (данные, информация, для хранения значений)
               - keyboard (клавиатура, пользовательский ввод)
         - Output (вывод результата вычислений или действий)
             - Sample (пример, образец, результат выполнения программы)
               - Variable (переменная)

               - Bidirectional isolation (изоляция двунаправленности, изоляция направления текста)
               - Bidirectional override (переопределение двунаправленности, смена направления текста)

               - Subscript (подстрочный индекс, для формул)
               - Superscript (верхний индекс, для формул)

             - Сite (цитировать, источник цитаты)
                   - Quote (цитата внутри абзаца)
